<b>NoWakeLock</b> - Advanced Android Power Management

Take full control of your Android device's power consumption by managing Wakelocks, Alarms, and Services with precision and ease.

<b>Why NoWakeLock?</b>
While modern Android's Doze mode helps with battery optimization, power users need granular control over background activities. NoWakeLock provides the tools to identify and manage power-hungry components that drain your battery.

<b>Key Features</b>

• <b>Complete Component Control</b> - Monitor and manage Wakelocks, Alarms, and Services
• <b>Smart Blocking</b> - Allow, limit, or block components with flexible rules
• <b>Regex Pattern Matching</b> - Create powerful rules using regular expressions
• <b>Material Design 3 UI</b> - Modern, intuitive interface built with Jetpack Compose
• <b>Detailed Statistics</b> - Track battery impact and optimization results
• <b>Multi-User Support</b> - Full compatibility with Android's multi-user environment
• <b>Module Status Check</b> - Verify Xposed framework integration
• <b>Backup & Restore</b> - Save and restore your configurations
• <b>App-Level Management</b> - Configure rules per application
• <b>Real-time Monitoring</b> - See live component activity

<b>Compatibility</b>
• Android 7.0 (API 24) to Android 15 (API 35)
• Requires Xposed Framework (LSPosed or EdXposed)
• Supports ARM64 and ARM32 architectures

<b>Open Source & Privacy</b>
• Fully open source (GPL v3.0)
• No data collection or tracking
• All processing happens locally on your device

<b>Important Notes</b>
• Requires root access and Xposed framework
• Beta software - use with caution
• Advanced users recommended
• May affect app functionality if misconfigured

Perfect for users who want maximum control over their device's power consumption and background activity.
